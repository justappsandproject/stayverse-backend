# STAYVERSE

Monorepo-style workspace containing the Stayverse backend, web apps, and mobile agents.

## Documentation

- **[Platform architecture and local development](docs/PLATFORM_LOCAL_DEVELOPMENT.md)** — how the apps connect, environment variables, ports, and run commands.

## Projects (short)

| Folder | Description |
|--------|-------------|
| `stayverse-backend-main` | NestJS API (MongoDB). |
| `stayverse-admin-main` | Vite + React admin dashboard. |
| `stayverse-agent-app-master` | Flutter agent app. |
| `stayverse-website-main` | Next.js marketing / landing site (Supabase for forms). |
| `stayverse-user-app-master` | Flutter user app (same API family as the agent app). |
